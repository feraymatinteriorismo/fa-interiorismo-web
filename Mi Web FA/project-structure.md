# Estructura del Proyecto - FA Interiorismo

## Archivos Principales
- **index.html**: Página principal con todas las secciones
- **portfolio.html**: Página dedicada a proyectos
- **servicios.html**: Detalles de servicios
- **main.js**: Funcionalidad y animaciones
- **styles.css**: Estilos personalizados (opcional, puede usar Tailwind)

## Secciones del Index
1. **Header/Navigation**: Logo, menú principal
2. **Hero Section**: Imagen de fondo, texto principal, CTA buttons
3. **Sobre Mí**: Bio de Fernanda, filosofía de trabajo
4. **Servicios**: Asesorías, proyectos integrales, renders 3D
5. **Proceso de Trabajo**: 4 pasos del proceso
6. **Portfolio**: Galería de proyectos con grid limpio
7. **Testimonios**: Reseñas de clientes
8. **Contacto**: Formulario y CTA final
9. **Footer**: Información de contacto y redes

## Recursos Necesarios
- Imagen hero: Render realista de interior cálido
- 12-15 imágenes de portfolio (proyectos y renders)
- Iconos para servicios
- Foto de Fernanda (opcional, puede ser ilustración)
- Textos profesionales y claros

## Funcionalidades
- Scroll suave entre secciones
- Formulario de contacto funcional
- Galería de imágenes con lightbox
- Responsive design
- Optimización SEO básica
- Carga rápida de imágenes
