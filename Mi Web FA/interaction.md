# Interacciones - FA Interiorismo

## Navegación
- Header fijo con navegación suave
- Menú responsive con transiciones
- Logo clickable que lleva al inicio
- Indicador de sección activa

## Botones Principales
1. **Ver proyectos**: Scroll suave a sección portfolio
2. **Solicitar asesoría**: Abre formulario de contacto modal
3. **WhatsApp**: Enlace directo con mensaje predefinido

## Formularios
- Formulario de contacto con validación
- Campos: Nombre, Email, Teléfono, Tipo de proyecto, Mensaje
- Confirmación de envío con animación
- Integración con WhatsApp para respuesta rápida

## Galería Portfolio
- Grid responsive de imágenes
- Lightbox para vista ampliada
- Filtros por tipo de proyecto (opcional)
- Lazy loading para optimización

## Scroll Animations
- Reveal de secciones al hacer scroll
- Parallax sutil en imágenes hero
- Counters animados en estadísticas
- Progreso de lectura en páginas largas

## Interacciones Mobile
- Touch gestures en galería
- Menú hamburguesa fluido
- Formularios optimizados para mobile
- Botones de tamaño adecuado para touch
